# Week 1 Lab Answers

## 1. What is ROS 2?

ROS 2 (Robot Operating System 2) is a framework used for developing robot software.  
It provides tools, libraries, and conventions to simplify the task of creating complex robot behavior.

## 2. What is a ROS 2 Node?

A node is a single executable that performs a specific task.  
Multiple nodes communicate with each other to create a robot system.

## 3. What is a ROS 2 Package?

A package is a directory that contains ROS 2 code, libraries, configuration files, and resources required for a project.

## 4. What does `colcon build` do?

`colcon build` compiles and builds ROS 2 packages inside a workspace.

## 5. What does `ros2 run` do?

`ros2 run` is used to execute a node from a specific ROS 2 package.
